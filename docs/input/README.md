# Truener
〜SNS趣味共鳴マッチングアプリ〜

## 概要
X（旧Twitter）・InstagramのSNSデータをAIで解析し、ユーザーの趣味・嗜好タグを自動生成。
共通の趣味を持つ相手とのマッチング精度を高め、「話のきっかけがある出会い」を提供する婚活向けWebアプリ。

## 解決する課題
結婚を真剣に考えているマッチングアプリ経験者が抱える「この人と本当に話が合うのかわからない」という不安と、
マッチング後に会話が続かず自然消滅してしまう問題を、SNS由来の共通点情報で解消する。

## ターゲットユーザー
25〜35歳、結婚を真剣に考えているマッチングアプリ経験者（男女）

## 主要機能
- SNS（X/Instagram）連携による趣味タグ自動生成
- 共通タグ数・相性スコアによるマッチング
- 共通点をハイライト表示するプロフィール画面
- AIによる最初のメッセージ提案（アイスブレイカー）
- Google認証によるシンプルな本人確認

## 料金体系
- 男性：月額6,000円（20いいねまで無料。メッセージ送信・21いいね目以降は課金必要）
- 女性：完全無料

## 女性向け特徴
- **女性ファーストメッセージ制：** マッチング後、最初のメッセージは女性からのみ送信可能（Bumble方式）
- **いいねリクエスト承認制：** 男性からのいいねはリクエストとして届き、女性が承認した相手とのみチャット開始
