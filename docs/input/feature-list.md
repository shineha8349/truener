## 機能詳細

### AIアイスブレイカー提案
**概要：** チャット開始時、共通タグをもとにAIが最初のメッセージ候補を複数提案する機能

**ユーザーストーリー：**
「マッチングアプリを使うユーザーとして、最初のメッセージに悩まずに会話を始めたい。なぜなら何を話せばいいかわからず、せっかくのマッチングを無駄にしてしまうから。」

**入力：**
- 自分の趣味タグ
- 相手の趣味タグ
- 共通タグ一覧

**出力：**
- 最初のメッセージ候補（2〜3パターン）
- カジュアルで親しみやすいトーン

**受け入れ条件：**
- [ ] チャット開始時に自動でアイスブレイカー提案が表示される
- [ ] 提案は共通タグに基づいた内容になっている
- [ ] ユーザーは提案をそのまま送信 or 編集して送信できる
- [ ] 提案を使わずに自分でメッセージを入力することもできる

---

### SNSタグ自動生成
**概要：** X/InstagramのAPIを通じて取得したデータをAIで解析し、ユーザーの趣味・嗜好タグを自動生成する機能

**ユーザーストーリー：**
「マッチングアプリに登録するユーザーとして、自分の趣味を手動で入力せずに自動でタグを設定してほしい。なぜならプロフィール作成が面倒で、本当の自分の趣味をうまく言語化できないから。」

**入力：**
- X投稿テキスト・いいね履歴・フォロー情報
- Instagram投稿・いいね履歴

**出力：**
- 趣味タグ一覧（例：「音楽好き」「ライブ好き」「カフェ巡り」「登山」「映画好き」）

**受け入れ条件：**
- [ ] SNS連携後、30秒以内にタグ生成が完了する
- [ ] 生成されたタグはユーザーが確認・修正できる
- [ ] SNS未連携でも手動でタグを追加できる
- [ ] タグは10〜20個程度が目安
