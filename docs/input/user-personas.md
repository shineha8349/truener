# ターゲットユーザー

## メインペルソナ①（女性）

### 基本情報
- 名前（仮）：田中 さくら
- 年齢：28歳
- 職業：IT企業の総合職（マーケティング担当）
- 生活環境：東京都内在住、一人暮らし

### 1日の流れ
- 平日：9時出社、仕事中はSlack・Notionを使いこなす。夜は自炊またはデリバリー
- 休日：ライブやカフェ巡りが好き。InstagramとXを合計1〜2時間ほど閲覧・投稿
- マッチングアプリは就寝前にチェックする習慣がある（1日15〜30分程度）

### 課題・ペイン
- Pairsを1年間使ったが「マッチングしても会話が3往復で終わる」ことに疲れた
- 会ってみると「全然違う」と感じることが多く、時間を無駄にしている感覚がある
- 「また同じ自己紹介をしなければいけない」という消耗感がある
- 「この人、本当に結婚を考えているのか？」という疑念が常にある

### 現在の解決方法
- 複数のマッチングアプリを並行して使用
- 友人に紹介を頼む（でもなかなか良い縁がない）

### 求めているもの
- 「話が合いそう」と最初からわかる相手との出会い
- 最初のメッセージに困らない、話しかけやすい仕組み
- 真剣な人だけが集まっている安心感

### このプロダクトを使う瞬間
- 夜、スマホでプロフィールをチェックするとき「この人と私、ライブ好き繋がりがある！」と気づいた瞬間
- マッチング後、AIが「共通点：音楽好き・カフェ巡り」をもとにメッセージ例を出してくれたとき

### 情報収集の方法
- Instagram・X・TikTokで口コミや体験談を見る
- 友人からの口コミ

## メインペルソナ②（男性）

### 基本情報
- 名前（仮）：鈴木 健太
- 年齢：31歳
- 職業：Web系ベンチャー企業のエンジニア
- 生活環境：神奈川県在住、一人暮らし

### 1日の流れ
- 平日：リモートワーク中心。Xを情報収集に日常的に使用
- 休日：登山・ランニング・映画鑑賞。Instagramに趣味の写真を投稿
- マッチングアプリは移動中や昼休みにチェック

### 課題・ペイン
- 「何を話せばいいかわからない」ため最初のメッセージを送ることを躊躇してしまう
- 自己紹介文を書くのが苦手で、プロフィールで自分をうまく表現できていないと感じる
- マッチングできても「お互いに何もわからない状態」から始まるのがしんどい

### 現在の解決方法
- withやOmiaiを利用。性格診断などを活用して相性を確認している

### 求めているもの
- 最初のメッセージのきっかけが自然に作られる仕組み
- 「この人は自分と趣味が合う」と確信を持って話しかけられる状況
- 真剣に婚活している女性と出会いたい

### このプロダクトを使う瞬間
- SNS連携後「あなたとの共通点：登山・映画・カフェ巡り」が表示された相手を見たとき
- AIが「〇〇さんも登山好きみたいですね！先週どこか行きましたか？」と提案してくれたとき

### 情報収集の方法
- Xのテック系アカウントや婚活系アカウントをフォロー
- YouTubeのマッチングアプリ解説動画

## 非ターゲット
- 婚活に真剣でない、またはカジュアルな出会いを求めているユーザー
- SNSアカウントを持っていない・非アクティブなユーザー（SNS連携の価値を享受できない）
- 結婚より交際優先で考えているユーザー
- 40代以上のユーザー（初期フェーズでは対象外。将来的に検討）
