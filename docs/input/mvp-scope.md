# MVP範囲

## MVP定義

### 目的
「SNS趣味タグによる共通点マッチング」というコアバリューが実際にユーザーに刺さるかを検証する。
具体的には、SNS連携タグをもとにしたマッチングによって、マッチング後の会話継続率が従来のマッチングアプリより向上するかを確認する。

### 対象ユーザー
- 25〜35歳、結婚を真剣に考えているマッチングアプリ経験者
- XまたはInstagramをアクティブに使っているユーザー
- 初期はクローズドβ（招待制）で少人数から開始

### 成功の定義
- マッチング後、3往復以上のチャットが発生した割合が50%以上
- SNS連携率が登録ユーザーの60%以上
- 1ヶ月継続率が40%以上

## スコープ

### MVP に含める ✅

| 機能 | 理由 |
|------|------|
| Google認証ログイン | シンプルな登録体験。メール登録より離脱が少ない |
| **年齢確認（公的証明書アップロード）** | **出会い系サイト規制法の遵守のためMVP必須。18歳未満の利用を排除** |
| 基本プロフィール作成（写真・ニックネーム・自己紹介） | マッチングの前提として必須 |
| SNS連携（**Instagram**のみ） | コアバリューの根幹。X APIはコスト面でMVP除外 |
| AIタグ自動生成 | SNS連携の価値を実現する機能 |
| タグ手動修正・追加 | タグ精度への不満をユーザー自身で解消できる（SNS未連携者のフォールバックも兼ねる） |
| マッチング候補一覧（共通タグ数・相性スコア表示） | コアバリューの体験に必須 |
| 詳細プロフィール（共通点ハイライト表示） | 「この人と話が合う」という確信を与える |
| いいねリクエスト承認制・マッチング成立通知 | マッチングの基本動作として必須 |
| 1対1チャット機能 | 出会いの目的達成に必須 |
| 女性ファーストメッセージ制 | 女性ユーザー獲得・定着のための必須施策 |
| AIアイスブレイカー提案 | コアバリューの「会話継続率向上」に直結 |
| ブロック・通報機能 | 最低限の安全性確保 |
| 男性月額課金（6,000円）・女性完全無料 | 収益確保・真剣ユーザーの担保・女性ユーザー母数最大化 |

### MVP に含めない ❌

| 機能 | 理由 | 対応時期 |
|------|------|----------|
| **X（旧Twitter）API連携** | **APIコストが高くMVP段階では許容できない** | **Phase 2** |
| クレジットカード連携認証 | 登録ハードルになる可能性。MVP後の信頼性向上施策として | Phase 2 |
| 年収・勤務先証明アップロード | 同上 | Phase 2 |
| タグ・条件フィルター検索 | MVP時点ではユーザー数が少ないため不要 | Phase 2 |
| 画像送信（チャット内） | テキストのみでも会話体験は成立する | Phase 2 |
| スワイプUI | カードリスト形式で代替可能 | 将来検討 |
| プッシュ通知（PWA） | ブラウザ通知で代替可能 | 将来検討 |
| AIによる詳細相性レポート（有料） | コアバリュー検証後に追加 | Phase 3 |
| スタンプ・絵文字リアクション | なくても会話は成立する | 将来検討 |

## タイムライン（目安）

| フェーズ | 期間 | 内容 |
|----------|------|------|
| MVP開発 | 〜3ヶ月 | Google認証・SNS連携・タグ生成・マッチング・チャットの実装 |
| クローズドβ | 1〜2ヶ月 | 招待制で50〜100名に先行利用してもらいフィードバック収集 |
| Phase 2 | 4〜6ヶ月目 | タグ精度改善・クレカ連携認証・フィルター機能・画像送信追加 |
| Phase 3 | 7ヶ月目〜 | 有料オプション追加・マーケティング本格化・ユーザー数拡大 |

---

## 制約事項
- Instagram Graph API の利用制限・コストへの対応が必要（X APIはPhase 2以降）
- **出会い系サイト規制法に基づく年齢確認の実装が法的義務**
- 個人情報保護法・プライバシーポリシーの整備が必須（年齢確認書類の取り扱いを含む）
- 初期はユーザー数が少ないため、マッチング数を確保するために性別バランス管理が重要

---

## リスクと対策

| リスク | 対策 |
|--------|------|
| SNS API仕様変更・停止 | 複数SNS対応・SNS未連携でも使える手動タグ入力を併用 |
| タグ精度が低く、ユーザーが不満を持つ | 手動修正機能を提供・フィードバックを収集してAIを改善 |
| 女性ユーザーが集まらず男性とのバランスが崩れる | 女性料金の見直し・招待制による初期バランス調整 |
| マッチング後も会話が続かない | アイスブレイカーの質を継続改善・チャット継続率をKPIとして追う |
