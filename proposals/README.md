# プロダクト提案

## 元のアイデア

結婚を真剣に考えている男女（マッチングアプリ経験者）をターゲットにした婚活マッチングアプリ。
X（旧Twitter）やInstagramと連携してユーザーの趣味嗜好を自動タグ付けし、共通点ベースのマッチング精度を向上させる。
さらにクレジットカード連携・位置情報を活用して身元の信頼性を担保することで、「この人は本当に信用できるのか」という不安を解消する。
料金体系は男性6,000円／月、女性2,000円／月。Webアプリとして提供。
