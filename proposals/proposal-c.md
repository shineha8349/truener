# 提案C：婚活コミュニティ×マッチング融合型
〜オンラインからオフラインへの自然な流れ〜

## コンセプト
「1対1のチャットだけじゃ、本当の相性はわからない」という課題に向き合い、
共通の趣味を持つグループコミュニティと、そこからの自然なマッチングを組み合わせたWebアプリ。
SNS趣味タグを活用したグループ参加から始まり、複数人での交流を経て1対1の関係へと発展させる。

## 解決する課題
- **課題の主体：** マッチングアプリ疲れを起こしている経験者
- **課題の内容：**
  - 1対1チャットの繰り返しに疲れた（「毎回同じ自己紹介が辛い」）
  - 会う前に相手の雰囲気がわからず、会ってみたら全然違った
  - 真剣に婚活しているのに、会話が続かず自然消滅することが多い

## ターゲットユーザー
- 28〜38歳、結婚を真剣に考えている男女
- マッチングアプリを複数回経験済みで「消耗した」と感じている人
- グループ活動や共通の趣味を通じた自然な出会いを求めている人
- SNSをアクティブに使っていてコミュニティ参加に抵抗がない人

## 主要機能
1. **SNS趣味タグ自動生成** — X/Instagramの投稿を分析し趣味タグを生成
2. **趣味コミュニティ機能** — タグをもとに「音楽好きグループ」「旅行好きグループ」などに参加できる場
3. **グループ内での自然な交流** — グループ内で気になった人に「いいね」を送る
4. **クレジットカード連携本人確認** — 参加資格として認証を必須化し、安全なコミュニティを維持
5. **1対1チャットへの移行** — グループで繋がった後、個別チャットへスムーズに移行

## プロダクト形態
Webアプリ（スマートフォンブラウザ最適化）

## ビジネスモデル案
- 男性：月額6,000円
- 女性：月額2,000円
- グループ参加数の上限を設定し、上限解放は有料オプションとして提供
※これは仮の方向性です。要検討。

## この提案の特徴
- **良い点：**
  - 1対1チャット偏重のマッチングアプリとの明確な差別化
  - グループ交流を通じて相手の人柄が事前にわかる
  - マッチングアプリ経験者の「疲れ」に直接応えるコンセプト
- **課題・リスク：**
  - コミュニティの活性化が必要で、初期ユーザー獲得が最大の壁
  - グループ機能の開発コストが他の提案より高い
  - 「真剣に婚活したい」ユーザーに「グループ活動」が刺さるかは要検証

## 想定されるユーザーフロー
1. メールアドレスで仮登録
2. クレジットカード情報を入力（本人確認 + 課金設定）
3. SNS（X/Instagram）連携 → 趣味タグが自動生成される
4. プロフィール入力・写真登録
5. 趣味タグに合ったコミュニティ（グループ）に参加
6. グループ内の投稿・リアクションを通じて交流
7. 気になった人に「いいね」→ マッチング → 1対1チャット開始

## 参考になりそうなサービス

- Facebook Dating（グループ機能との連携）
- Bumble BFF（友達作りコミュニティからの発展という発想）
